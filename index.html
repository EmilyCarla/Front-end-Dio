Títulos (h1 a h6)

Os títulos em HTML são usados para estruturar o conteúdo de uma página e indicar a hierarquia das informações. Existem seis níveis de títulos, do h1 (mais importante) ao h6 (menos importante).

Exemplos de uso:

  1. Este é um exemplo de título h1

  2. Este é um exemplo de título h2

  3. Este é um exemplo de título h3

  4. Este é um exemplo de título h4

  5. Este é um exemplo de título h5
  6. Este é um exemplo de título h6

Tags de listagem

As listas em HTML são usadas para organizar informações de forma estruturada. Elas ajudam a exibir conteúdos de forma ordenada ou não ordenada, facilitando a leitura e a compreensão.

  • Listas não ordenadas "ul": Utilizam marcadores (pontos, círculos, quadrados) para apresentar os itens sem uma sequência específica. Como no exemplo de uma lista de compras:
    • Arroz
    • Feijão
    • Óleo
  • Listas ordenadas "ol": Os itens são numerados automaticamente, indicando uma ordem ou hierarquia. Exemplo:
    1. Primeiro lugar
    2. Segundo lugar
    3. Terceiro lugar

Destaques no texto

Essas tags são utilizadas para dar destaque a partes do texto, como, por exemplo, deixar em negrito, itálico ou grifado.

Alguns exemplos são as tags "small" que exibe o texto em um tamanho menor pequeno dessa maneira. A tag "i" que deixa o texto em itálico dessa forma, a tag "u" que sublinha o texto como nesse exemplo, a tag "strong" que da ênfase ao texto, além de aumentar sua semântica, o deixando dessa maneira, e a tag "mark" que faz com que o texto fique grifado como nos subtitulos dessa página.

Uso de citações

São usadas para evidenciar citações no texto, um exemplo é o "blockquote" que deixa a frase na linha seguinte, como no exemplo:

"A melhor maneira de prever o futuro é inventá-lo - Nikola Tesla".
"Só sei, que nada sei" - Sócrates

Textos com formatação especial

Dois exemplos são as tags "sub" e "sup" usadas para sobrescrever e subscrever, respectivamente. São muito úteis em fórmulas matemáticas. Exemplos:

x2+4=8

7x+y=21

Links e separação de conteúdo

São usadas para organizar a maneira de separar os conteúdos da página, como por exemplo a tag "hr" que cria uma linha horizontal entre os tópicos, como usado nessa página, e também a tag "a" que cria um link com uma referência para algum lugar predefinido, também usada no índice dessa página.

Outras

Seção para exemplificar algumas tags extras como, por exemplo: "font", "del" e "abbr".

A tag font é usada para estilizar o texto, usando os atributos cor, tamanho e fonte, o deixando como nesse exemplo: Este texto está em vermelho, tamanho 3 e fonte Arial, totalmente estilizado pela tag "font".

Essa estilização também pode ser feita usando css.

A tag "del" indica que um texto foi excluído ou que não é mais válido, ficando rasurado. Exemplo:

Promoção! O preço original era R$ 100,00, mas agora está por R$ 80,00!

A tag "abbr" tem a função de definir abreviações ou siglas, exibindo um texto explicativo quando o usuário passa o mouse sobre ela. Exemplo:

- "A ONU foi fundada em 1945".

Dessa maneira, quando o usuário passa o mouse sobre a palavra "ONU", aparece a explicação.
